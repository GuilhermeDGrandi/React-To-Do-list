# 📝 To-Do List

Um simples aplicativo de lista de tarefas feito em **React** para praticar conceitos fundamentais da biblioteca.

## 🚀 Tecnologias Utilizadas
- React.js ⚛️
- JavaScript (JSX)
- HTML5
- CSS3

## 🎯 Funcionalidades
✅ Adicionar tarefas  
✅ Editar tarefas  
✅ Marcar como concluída  
✅ Remover tarefas
